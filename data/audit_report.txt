NOTABLE OBSERVATIONS
================================================================================

1. SMART ERROR PAGE HANDLING: The pipeline correctly distinguishes between
   real files and HTML error pages served with HTTP 200 status codes. Several
   sites (moef.go.kr, khs.go.kr, seoul.go.kr, nec.go.kr, kma.go.kr,
   scourt.go.kr, incheon.scourt.go.kr, jbe.go.kr) return HTTP 200 for
   nonexistent files but serve HTML error pages. The pipeline correctly
   identifies these as "not existing."

2. SNAPSHOT FIDELITY: All 19 robots.txt snapshots (excluding M17 which has
   none) match the live content exactly, confirming the data was captured
   accurately.

3. NUANCED BLOCKING DETECTION:
   - Seoul (L01): Correctly identified as "partial_allow" despite starting
     with "Disallow: /" because Allow directives override for specific paths.
   - Jeonbuk Education (E013): Correctly notes Googlebot is "blocked" while
     LLM crawlers are "allowed" (not mentioned = allowed).
   - Supreme Court (J001): AhrefsBot and SemrushBot are fully blocked, but
     this is correctly not flagged as "full_block" since the * rule only
     partially blocks.

4. UNREACHABLE SITES: M17 (molit.go.kr) has SSL issues (curl exit 35),
   and the pipeline correctly records it as having no robots.txt, llms.txt,
   or sitemap.xml, with main_page_status: 0.
